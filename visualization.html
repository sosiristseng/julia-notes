Visualization#

Plotting libraries for statistical Charts, Graphs, Plots, Histograms, Maps.

  • Julia Plots Organization

Plotting#

  • ASCIIPlots.jl : Generate simple plots as ASCII art in Julia.

  • BlenderPlot.jl : a prototype version of the bar_chart plot on Blender using PyCall.

  • Bokeh.jl: Julia bindings for the Bokeh plotting library. The gallery.

  • CMDimCircuits.jl : Parametric analysis/visualization of model/measurement/simulation results. The sister package for continuous function interpolation: CMDimData.jl.

  • CMPlot.jl : Cloudy Mountain Plot inspired by Violin, Bean and Pirate Plots.

  • CrossfilterCharts.jl : A Julia package for automagical DC.js linked charts in your IJulia notebook.

  • DataVoyager.jl : Julia wrapper for the Voyager data exploration tool.

  • Displaz.jl : Julia bindings for lidar viewer displaz.

  • ECharts.jl: Julia package for the Apache ECharts v4 visualization library.

  • Gadfly.jl : A statistical graphics plotting and data visualization system. Gadfly Reference Cards and rendering to an SVG file using the D3 browser interface.

  • Gaston.jl : A julia front-end for gnuplot.

  • GMT.jl : is a package containing Julia bindings for the Generic Mapping Tools Library.

  • GR.jl : This module provides a Julia interface to the GR framework graphics library.

  • GracePlot.jl : Publication-quality plotting for julia using Grace/xmgrace.

  • GraphMakie.jl : Plotting graphs with Makie.jl.

  • GraphPlot.jl : Graph visualization for Julia.

  • Immerse.jl : Dive deeper into your data with interactive graphics.

  • ImPlot.jl : an interface to cimplot, which is an auto-generated C API to implot, a C++ plotting extension library for imgui.

  • Javis.jl: Julia Animations and Visualizations. JuliaCon 2021 Video.

  • Makie.jl : High level plotting on the GPU. Examples could be found at BeautifulMakie.

  • PGFPlots.jl : Plotting tool that uses the LaTeX pgfplots package (built on top of TikZ) to produce plots.

  • PGFPlotsX.jl : Generate publication quality figures using the LaTeX library PGFPlots. It tries to have a very close mapping to the PGFPlots API as well as minimize the number of dependencies.

  • Plotly.jl : A Julia wrapper for the plot.ly REST API cloud service.

  • PlotlyJS.jl : Julia library for plotting with plotly.js.

  • Plots.jl : An API/interface and wrapper that sits above other plotting packages (backends) and gives the user simple, consistent, and flexible plotting commands.

  • PyPlot.jl : Plotting for Julia based on matplotlib.pyplot.

  • RoMEPlotting.jl: 2D plotting functionality for the RoME.jl package. It‚Äôs part of the Caesar.jl package.

  • SandDance.jl : Julia bindings for MS Sanddance

  • SimplePlots.jl : Plots in 5 seconds or your money back.

  • StatsPlots.jl : Statistical plotting recipes for Plots.jl.

  • UnicodePlots.jl : Unicode plotting for the REPL in Julia.

  • VegaLite.jl : This package provides access to the Vega-Lite high-level visualization grammar in Julia.

  • VennEuler.jl : Generate area-proportional Venn/Euler diagrams in Julia.

  • Winston.jl : 2D plotting for Julia.

Base packages#

  • Cairo.jl : Bindings for the Cairo graphics library.

  • Colors.jl : Color manipulation utilities for Julia.

  • ColorTypes.jl : Color primitives for Julia, based on FixedSizeArrays.

  • ColorVectorSpace.jl : This package is an add-on to ColorTypes, and provides fast mathematical operations for objects with types such as RGB and Gray - Treat colors as if they are n-vectors for the purposes of arithmetic.

  • GLFW.jl : Julia interface to GLFW, a multi-platform library for creating windows with OpenGL contexts and managing input and events.

  • GraphRecipes.jl : Graph-related recipes to be used with Plots.jl.

  • ImplicitEquations.jl : Plotting implicit equations and inequalities.

  • PlotlyBase.jl : Generate plotly.js JSON in Julia.

  • PlotRecipes.jl : Assorted recipes to be used with Plots.jl.

  • PlotThemes.jl : Themes for the Julia plotting package Plots.jl.

  • PlotUtils.jl : Generic helper algorithms for building plotting components.

  • RecipesBase.jl : Base package for defining transformation recipes on user types for Plots.jl.
