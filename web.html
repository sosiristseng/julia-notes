Web technology#

Generic external servers, web framework, etc., in Julia.

  • Julia Web Organization

HTML and CSS#

  • Cascadia.jl : A CSS Selector library in Julia.

  • DisplayAs.jl : It provides functions to show objects in a chosen MIME type.

  • Hyperscript.jl : A lightweight DOM representation for working with HTML, SVG, and CSS in Julia.

  • Kelpie.jl : an HTML templating engine in Julia.

Email#

  • SMTPClient.jl : An SMTP client to send emails from Julia.

Server backends#

  • Canvas.jl (by @fredrikekre) : a Julia package for interacting with the Canvas LMS REST API, which is a online education platform.

  • Discord.jl : Julia Discord wrapper.

  • HoJBot.jl : The Discord bot for Humans of Julia community server.

  • Joseki.jl : Suggested opening moves for building APIs in Julia.

  • NewsAPI.jl : Helper functions for using newsapi.org

  • Telegram.jl : Native Julia Telegram bot api wrapper.

  • Twitter.jl : Julia package to access the Twitter API.

Web frameworks#

  • Bukdu.jl : A web development framework for Julia, influenced by Phoenix framework.

  • Genie.jl : The highly productive Julia web framework.

  • Merly.jl : A micro framework for web programming in Julia.

Middleware#

  • JuliaWebAPI.jl : Julia package for deploying APIs on JuliaBox to facilitat wrapping Julia functions into a remote callable API via ZMQ and HTTP.

  • Mux.jl : Middleware for Julia.

  • ZMQ.jl : Julia interface to ZeroMQ.

Networking#

  • DandelionWebSockets.jl : A Julia package for client side WebSockets.

  • IPNets.jl : IPv4 / IPv6 network abstractions for Julia.

  • LogParser.jl : A package for parsing server logs. Currently, only server logs having the Apache Combined format are supported (although Apache Common may parse as well).

  • Rate Limiter.jl : Julia package for limiting the rate at which events occur.

  • TimeZones.jl : Olsen Timezone Database Access for the Julia Programming Language.

Security#

  • GnuTLS.jl : Transport Level Security for Julia Streams provided by GnuTLS.

  • MbedTLS : Wrapper around mbedtls.

  • OAuth.jl : Pure Julia implementation of OAuth v1.0a.

Web#

  • Blink.jl : Web-based GUIs for Julia.

  • CSSUtil : utilities to create and align various web elements on the DOM.

  • CutShortURL.jl : The simplest URL Shortener (tinyurl) for Julia.

  • DandelionWebSockets.jl : client-side WebSockets.

  • Dash.jl : A Julia interface to the Dash ecosystem for creating analytic web applications in Julia.

  • Diana.jl : GraphQL for Julia.

  • Escher.jl : Composable Web UIs in pure Julia. Website.

  • Gumbo.jl : Julia wrapper around google’s gumbo library for parsing HTML.

  • HTTP.jl : HTTP for Julia.

  • HTTP2.jl : A HTTP2 support library for Julia. Only use this library directly if you need low-level functionality.

  • HttpCommon.jl : Provides types and helper functions for dealing with the HTTP protocol in Julia.

  • LibCURL.jl : Thin Julia wrapper of libCURL

  • Pages.jl : A simple way to create and interact with web pages.

  • Skans.jl : Monitor web pages and get notified when a page has changed.

  • UAParser.jl : UAParser is a Julia port of ua-parser, which itself is a multi-language port of BrowserScope’s user agent string parser.

  • URIs.jl : URI parsing in Julia

  • URITemplate.jl : This package provides URI Template interpolation by implementing (RFC 6570).

  • WebIO.jl : WebIO provides a simple abstraction for displaying and interacting with web content.

  • WebSockets.jl : A WebSockets server library for Julia.

Static site generators#

  • Franklin.jl : static site generation with live Julia code evaulation. See also examples and templates

  • PkgPage.jl : package front-pages powered by Franklin.jl.

  • PlutoSliderServer.jl : Web server to host Pluto notebooks supporting static conversion to HTML files.

  • PlutoStaticHTML.jl : Convert Pluto notebooks to pure HTML files. Template for Julia tutorials.

  • StaticWebPages.jl : A black-box generator for static websites oriented toward academics and personal web-pages.
